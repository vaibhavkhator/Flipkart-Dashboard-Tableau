# 📊 Performance Marketing & Sales Dashboard (Tableau)

## 📌 Overview

This project presents a **Tableau dashboard** designed to analyze **advertising performance and product-level sales data**. It provides actionable insights into how marketing spend impacts revenue, conversion rates, and product growth.

The dashboard helps businesses make **data-driven decisions** by tracking key performance indicators such as ROAS, conversions, and budget utilization.

---

## 🎯 Objectives

* Analyze **ad performance** across different brands
* Track **conversion rates and user engagement**
* Evaluate **Return on Ad Spend (ROAS)**
* Identify **top-performing products (SKU level)**
* Monitor **budget vs actual ad spend**
* Understand **growth drivers in sales**

---

## 📂 Dataset Description

The dashboard is built using datasets containing:

### 🔹 Advertising Data

* Date
* Brand (BA, CAP, SS)
* Ad Spend
* Ad Sales
* Clicks
* Conversion Rate

### 🔹 Product Data

* SKU / Product Name
* Category
* Sales Value
* Quantity

### 🔹 Target Data

* Daily Ad Spend Target
* Monthly (MTD) Targets

---

## 📊 Key Features

### 1. Session & Conversion Analysis

* Dual-axis chart showing **Clicks vs Conversion Rate**
* Helps evaluate traffic quality and campaign effectiveness

### 2. ROAS (Return on Ad Spend)

* Brand-wise and daily ROAS breakdown
* Formula:

  ```
  ROAS = Ad Sales / Ad Spend
  ```

### 3. SKU-Level Analysis

* Top-performing products based on sales
* Helps identify best-selling SKUs

### 4. Growth Drivers

* Highlights products contributing most to revenue growth
* Shows absolute growth and percentage contribution

### 5. Sales Trend

* Daily sales visualization
* Tracks revenue fluctuations over time

### 6. Ads Spend vs Target

* Compares **actual spend vs expected spend**
* Helps monitor over/under-spending

### 7. MTD Budget Utilization

* Donut chart showing:

  * Total spend
  * Remaining budget
  * Percentage utilized

## ⚙️ Tools & Technologies

* **Tableau** – Dashboard creation and visualization
* **Excel/CSV** – Data source
* **Calculated Fields** – For KPIs like ROAS, Growth, Conversion

---

## 🛠️ How It Was Built

1. Imported datasets into Tableau
2. Cleaned and structured data
3. Created calculated fields:

   * ROAS
   * Conversion Rate
   * Growth & Contribution %
4. Built individual worksheets:

   * Charts, tables, and KPIs
5. Combined all sheets into a dashboard
6. Added filters for interactivity:

   * Brand
   * Category
   * Date
